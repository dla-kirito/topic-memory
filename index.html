Task context that
survives /compact.

/compact resets the context window — but wipes your task state too. Topic Memory hooks in to save decisions, preferences, and pitfalls, so you can pick up exactly where you left off.

install
$ curl -sSL https://raw.githubusercontent.com/dla-kirito/topic-memory/main/install.sh | bash

Requires Python 3 · Restart Claude Code after install

How it works

Step 01
Hooks into /compact
Every time you run /compact, Topic Memory automatically extracts the session's task context — no extra steps.
Step 02
Saved as a topic file
Decisions, preferences, and pitfalls — structured per project, persisted across sessions.
Step 03
Resume with a command
/topic-recall surfaces the right context. Claude ranks by relevance — you just pick and continue.
